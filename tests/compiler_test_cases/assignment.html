Test #1 Assignment
Description: Determine the bytecode after compilation of various assignments (int, char, string, float, bool).

Input: 

main()
{
    int i = 2;
    char c = 'c';
    float f = 1.1;
    string s = "string";
    bool b = true;
}

Expected/Actual Output:

  1. [2, 2, 140, :i, 5, "c", 140, :c, 3, 1.1, 140, :f, 4, "string", 140, :s, 1, true, 140, :b]
